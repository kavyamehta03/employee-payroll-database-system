# Employee Payroll Database System

## Technologies
SQL, MySQL

## Project Description
This project implements a normalized payroll database system used to manage employee information, salary records, and department data.

The database supports:
- Employee record management
- Payroll calculations
- Department-wise salary analysis
- Data integrity using primary and foreign keys

## Features
- Employee table management
- Payroll data tracking
- SQL joins and aggregate queries
- Salary and department analysis
